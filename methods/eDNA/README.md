# eDNA Methods

This folder contains Biomek i7 automated methods used for environmental DNA workflows at Minderoo OceanOmics centre at the University of Western Australia .

Each method should have its own folder containing the Biomek method file, supporting export files, documentation, screenshots and validation notes.

## Folder structure

Each method folder should use this structure:

- `method/` — Biomek method files.
- `exports/` — Biomek import/export files required to recreate the method environment.
- `docs/` — deck layout screenshots, guided setup screenshots and operator notes.
- `validation/` — simulation, water run and validation notes.
